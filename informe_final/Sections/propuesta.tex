
\subsection{Objetivo General}

El objetivo del proyecto es desarrollar e implementar un modelo de valoración de opcionalidades de prepago en créditos hipotecarios, utilizando modelos de tasas de interés ajustados al mercado chileno actual que permita cuantificar el costo de dicha opcionalidad para ser incorporado en la tasa de interés ofrecida al cliente.

\subsection{Objetivos Específicos}
\begin{enumerate}
    \item Caracterizar la estructura temporal de tasas de interés en el mercado chileno mediante el análisis de datos históricos y actuales.
    \item Implementar modelos de tasa corta, en particular el modelo de Ho--Lee, para generar árboles de tasas de interés consistentes con una curva inicial.
    \item Valorar la opción de prepago, modelándola como una opción americana sobre el crédito, determinando el \textit{spread} de opcionalidad necesario para compensar el riesgo asumido por el banco.
\end{enumerate}
